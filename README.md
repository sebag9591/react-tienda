# react-tienda

**react-tienda** es una tienda online **ficticia**, creada con fines académicos para practicar desarrollo web con **React**, **Vite** y **Bootstrap**. Este proyecto simula la experiencia de un e-commerce, permitiendo agregar y eliminar productos de un carrito, pero **no realiza ventas reales**.

---

## Proyecto

La aplicación simula el funcionamiento básico de una tienda online:

- Listado de productos
- Detalle de producto
- Carrito de compras
- Búsqueda de productos
- Sección de administración protegida por login

Los productos se obtienen desde una **MockAPI personalizada**, creada únicamente para este proyecto.

### Instalación local

1. Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/react-tienda.git
```
2. Entrar al directorio del proyecto:

```bash
cd react-tienda
```

3. Instalar dependencias
```bash
npm install
```

4. Ejecutar el proyecto
```bash
npm run dev
```

La aplicación se abrirá por defecto en:

```
http://localhost:5173
```
